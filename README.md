# Gerenciador de Livros (Spring Boot RESTful)

Este é um projeto de API RESTful para gerenciar livros, desenvolvido com Spring Boot. A ideia é ter um backend simples para operações de CRUD (Criar, Ler, Atualizar, Deletar) de recursos do tipo “Livro”.

---

## Funcionalidades

 - Cadastrar novo livro
 - Listar todos os livros
 - Buscar um livro por ID
 - Atualizar livro existente
 - Remover um livro
 - Validação básica de dados

---

## Tecnologias utilizadas

 - Java
 - Spring Boot
 - Spring Data JPA
 - Banco de dados (configurável — por exemplo, H2, MySQL, PostgreSQL)
 - Maven
